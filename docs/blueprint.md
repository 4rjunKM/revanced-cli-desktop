# **App Name**: VanceForge

## Core Features:

- Local APK Uploader: Easily drag and drop local Android APK files to be processed by the patcher interface.
- AI Patch Strategy Tool: An AI-powered tool that analyzes the provided APK version and suggests the most stable and useful patches based on current community feedback.
- Live Patching Terminal: A visual console showing real-time logs and progress of the ReVanced CLI execution for immediate feedback.
- Patch Management Hub: Browse and select from available official ReVanced patches with detailed descriptions and compatibility warnings.
- Automated CLI Integration: Bridges the web interface with local system commands to run the ReVanced patching operations efficiently.

## Style Guidelines:

- Primary color: Electric Indigo (#7C73EB), providing a high-tech and reliable appearance against a dark background.
- Background color: Deep Obsidian (#14121F), creating a focused environment for advanced technical operations.
- Accent color: Celestial Blue (#1982E6), used for active states and critical calls to action to ensure high visual contrast.
- Font pairing: 'Space Grotesk' for bold, technical headlines and 'Inter' for clear, legible body text and terminal logs.
- Utilize minimalist, sharp-edged monoline icons to maintain a sophisticated and modern software engineering vibe.
- A structured, bento-grid style dashboard that organizes APK stats, patch options, and live build logs into distinct, card-like regions.
- Micro-interactions on hover and progress bars that utilize smooth spring transitions for a polished desktop feel.
